# P0-04: Substation Dielectric Ground-Grid Retrofit — Complete Fabrication
## CSMFAB-P004 | IF-03 | Priority P0

**Function:** Per-acre EM hardening for electrical substations, replacing conductive copper grounding mesh with dielectric geopolymer + CNT-polymer ground rods that block GIC entry paths.

| Parameter | Value |
|-----------|-------|
| Coverage | 1 acre (4,047 m²) per unit |
| Geopolymer ground layer | CSM-MAT-GP-001, 150mm thick, 85 MPa |
| CNT-polymer ground rods | 4 × CNT-003, 25mm dia × 3m, driven at corners |
| FEATHER mesh nodes | 8 per acre (boundary current monitoring) |
| GIC path disruption | >10⁸ Ω from perimeter to equipment pads |
| Equipment pad isolation | ZTA ceramic bearing pads under each transformer |
| Conductor bonding | Existing copper bus retained but isolated via CNT-003 risers |
| Total installed cost | $340,000/acre |
| Annual maintenance | $4,200/acre (FEATHER node battery replacement) |
| Retrofit time | 3 weeks per acre (phased, substation remains energized) |

## INSTALLATION PROCEDURE
1. Excavate 150mm across acre, retaining existing copper ground mesh
2. Pour CSM-MAT-GP-001 geopolymer to grade, cure 7 days
3. Install ZTA equipment pads under each transformer
4. Drive CNT-003 ground rods at perimeter corners
5. Bond existing copper to CNT-003 risers with dielectric transition joints
6. Deploy 8× FEATHER mesh nodes
7. Test: DC resistance perimeter-to-equipment >100 MΩ at 1kV
