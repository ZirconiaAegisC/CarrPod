# INCURSION COMMON-11: Payload Bay Modular Interface Standard
## CSMFAB-INCURSION | Common Architecture | All 5 Vessel Classes

Standardized 200×150mm mounting plate with 4× M6 BFRP bolts. Power: 12V/5A + 5V/3A via CNT-polymer connector. Data: S-Bus over UART. Mechanical: IP68 when bay door sealed. Compatible payloads: GoPro housing, multibeam sonar, water sampler carousel, CarrPod Mini, medical cold box, plankton net, magnetometer tow body.

Applies to: I-SCOUT, I-COURIER, I-SURVEYOR, I-GUARDIAN, I-DEEP. Commonality ensures 70% parts sharing across fleet.
