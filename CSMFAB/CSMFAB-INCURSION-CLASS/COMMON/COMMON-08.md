# INCURSION COMMON-08: Navigation Sensor Suite — Common Components
## CSMFAB-INCURSION | Common Architecture | All 5 Vessel Classes

u-blox ZED-F9P RTK GPS (1cm accuracy). Bosch BMI270 IMU (6-axis). Honeywell HMC5983 magnetometer (GIC immune via Protonic-HFET interface). MS5837 pressure/depth sensor (300m rated). All sensors on isolated CNT-polymer data bus. No silicon in signal path — GaN ADC front-end.

Applies to: I-SCOUT, I-COURIER, I-SURVEYOR, I-GUARDIAN, I-DEEP. Commonality ensures 70% parts sharing across fleet.
