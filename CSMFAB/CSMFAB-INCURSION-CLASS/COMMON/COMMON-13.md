# INCURSION COMMON-13: Tardigrade Cryptobiotic Survival Mode
## CSMFAB-INCURSION | Common Architecture | All 5 Vessel Classes

In extreme conditions (storm, vessel strike risk, enemy detection): vessel enters 'tun state.' Hydrofoils retract. Ballast tanks flood to neutral buoyancy. All non-essential systems power down. Core-1 enters deep sleep (<1mW). FEATHER node transmits position once per hour. Vessel drifts passively. Survivable duration: 30 days (Scout) to 90 days (Deep). Reactivation: FEATHER wake command or pre-programmed timer.

Applies to: I-SCOUT, I-COURIER, I-SURVEYOR, I-GUARDIAN, I-DEEP. Commonality ensures 70% parts sharing across fleet.
