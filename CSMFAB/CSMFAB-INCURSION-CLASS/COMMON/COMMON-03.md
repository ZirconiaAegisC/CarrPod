# INCURSION COMMON-03: Retractable Hydrofoil System — Common Architecture
## CSMFAB-INCURSION | Common Architecture | All 5 Vessel Classes

2 forward canards + 2 aft main foils. ZTA foil sections with BFRP spar. MRF rotary actuator (CSM-MRF-001). Deployment time <2 sec. Retraction: hydrofoils fold flush into hull channels. Sealed by hydrogel gasket (tardigrade-derived swell material). Rated 50,000 cycles.

Applies to: I-SCOUT, I-COURIER, I-SURVEYOR, I-GUARDIAN, I-DEEP. Commonality ensures 70% parts sharing across fleet.
