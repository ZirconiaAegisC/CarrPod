# P0-01-VOL-15: Assembly Sequence — Stage 5: Retrofit of Existing Bridges
## CSMFAB-P001 | Ceramic Bridge Bearings 500T

---

## RETROFIT PROCEDURE — Steel-to-Ceramic Bearing Replacement

### Pre-Retrofit Assessment
| Item | Method | Data Collected |
|------|--------|---------------|
| Existing bearing type | Visual + design drawings | Steel-laminated elastomeric, pot, or disc |
| Bearing condition | Visual + ultrasonic | Cracking, corrosion, elastomer deterioration |
| Pedestal condition | Visual + Schmidt hammer | Concrete spalling, compressive strength |
| GIC path measurement | DC resistance test | Resistance superstructure-to-substructure |
| Traffic impact | Traffic count data | AADT, peak hour volume |

### Retrofit Sequence

| Step | Operation | Notes |
|------|-----------|-------|
| R1 | Install temporary support jacks (4× 200T hydraulic) | Positioned adjacent to bearing |
| R2 | Lift bridge 15mm | Controlled, monitored via LVDT |
| R3 | Remove old bearing | Cut anchor bolts, crane out |
| R4 | Chip out old grout pad | To sound concrete |
| R5 | Pour new geopolymer grout pad (CSM-MAT-GP-001) | Form, pour, cure 7d |
| R6 | Install ceramic bearing per Vol-13 procedure | Stage 3 field install |
| R7 | Lower bridge onto new bearing | Controlled descent, 1mm/min |
| R8 | Post-install verification per Vol-14 | Full checklist |
