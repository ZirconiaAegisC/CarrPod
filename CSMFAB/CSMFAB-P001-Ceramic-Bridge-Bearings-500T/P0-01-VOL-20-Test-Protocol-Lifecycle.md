# P0-01-VOL-20: Test Protocol — Lifecycle Endurance
## CSMFAB-P001 | Ceramic Bridge Bearings 500T

---

## LIFECYCLE ENDURANCE TEST BATTERY

### TEST L-01: Accelerated Weathering (QUV)

| Parameter | Value |
|-----------|-------|
| Standard | ASTM G154, Cycle 1 (UVA-340 lamps) |
| Exposure | 5,000 hours |
| Cycle | 8hr UV @ 60°C / 4hr condensation @ 50°C |
| Post-test | Visual, dielectric, PTFE friction coefficient |
| Pass criterion | No discoloration, dielectric unchanged, PTFE μ <0.06 |
| Frequency | First article only |

### TEST L-02: Salt Spray Corrosion

| Parameter | Value |
|-----------|-------|
| Standard | ASTM B117 |
| Duration | 2,000 hours |
| Solution | 5% NaCl, 35°C |
| Post-test | Visual + bolt torque re-check |
| Pass criterion | No corrosion on BFRP, bolt torque ±10% of initial |
| Frequency | First article only |

### TEST L-03: Combined Cycle Endurance

| Parameter | Value |
|-----------|-------|
| Cycles | Mechanical (100K cycles @ 50-500T) + Thermal (50 cycles -40/+85) + Wet/dry (25 cycles) |
| Sequence | All three imposed simultaneously in environmental chamber |
| Post-test | Full electrical + mechanical test battery |
| Pass criterion | All pass criteria maintained within 90% of initial values |
| Frequency | First article only |

### TEST L-04: PTFE Wear Life

| Parameter | Value |
|-----------|-------|
| Translation cycles | ±50mm, 500T vertical load, 0.01 Hz |
| Cycles | 50,000 |
| Measurement | PTFE thickness (ultrasonic), friction coefficient |
| Pass criterion | Wear depth <2mm, μ <0.08 |
| Equivalent service life | 150 years (at average 5mm/day bridge thermal movement) |

## SERVICE LIFE PREDICTION
Based on accelerated lifecycle testing, the predicted service life is **>100 years** with no maintenance required on the ceramic disc. PTFE sliding surfaces require inspection at 25-year intervals, with replacement at 50-75 years depending on traffic volume and thermal cycling severity.
