# P0-01-VOL-18: Test Protocol — Thermal
## CSMFAB-P001 | Ceramic Bridge Bearings 500T

---

## THERMAL TEST BATTERY

### TEST T-01: Thermal Shock (Factory Sample)

| Parameter | Value |
|-----------|-------|
| Cold soak | -40°C, 4 hours |
| Hot transfer | Into +85°C chamber, <10 sec transfer |
| Hot soak | +85°C, 4 hours |
| Cycles | 5 full cycles |
| Post-test | Dielectric test E-01 + visual crack inspection |
| Pass criterion | No cracking, dielectric unchanged |
| Frequency | 1 per 100 units |

### TEST T-02: Thermal Expansion Measurement

| Parameter | Value |
|-----------|-------|
| Temperature range | -40°C to +85°C |
| Measurement | LVDT on bearing height, 4 corners |
| Data collected | CTE of assembly (ceramic + BFRP housing) |
| Pass criterion | CTE within 9.8±1.0 ×10⁻⁶/°C (ceramic dominant) |
| Frequency | First article only |

### TEST T-03: Freeze-Thaw with De-Icing Salt

| Parameter | Value |
|-----------|-------|
| Solution | 3% NaCl, applied at +5°C |
| Cycle | +5°C/2hr → -20°C/4hr → +5°C/2hr |
| Cycles | 50 |
| Post-test | Visual inspection for spalling, dielectric test E-01 |
| Pass criterion | No spalling, dielectric unchanged |
| Frequency | 1 per 500 units |

### TEST T-04: Elevated Temperature Creep (65°C)

| Parameter | Value |
|-----------|-------|
| Temperature | 65°C |
| Vertical load | 500 tonnes |
| Duration | 1,000 hours |
| Measurement | Creep deflection (hourly log) |
| Pass criterion | Creep rate <0.1mm/100hr after initial settlement |
| Frequency | First article only |
