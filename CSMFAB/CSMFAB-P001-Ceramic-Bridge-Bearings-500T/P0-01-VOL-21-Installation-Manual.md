## QUICK REFERENCE — FIELD INSTALLATION

### Safety
- Hard hat, steel-toe boots, high-visibility vest MANDATORY
- Bearing weight: 32 kg — 2-person lift or crane
- Hydraulic jacks: inspect hoses and fittings before each use
- Traffic control per MUTCD Chapter 6F — lane closure minimum

### Critical Dimensions
| Parameter | Tolerance |
|-----------|-----------|
| Bearing position | ±3mm of design |
| Grout pad flatness | ±1mm over 350mm × 350mm |
| Anchor bolt torque | 450 N·m ±5% |
| Epoxy cure time | 4hr min @ 20°C (longer below 10°C) |
| Post-install settlement | ≤3mm, ≤1mm differential |
| Dielectric test | >100 MΩ @ 1kV |

### Common Errors
- **Overtorquing BFRP bolts:** >500 N·m strips threads. Use torque wrench, not impact gun.
- **Insufficient epoxy cure:** Loading before full cure cracks the bond. Verify temperature-adjusted cure time.
- **Grout pad unlevel:** Causes edge-loading on ceramic disc. Shim or re-pour.
