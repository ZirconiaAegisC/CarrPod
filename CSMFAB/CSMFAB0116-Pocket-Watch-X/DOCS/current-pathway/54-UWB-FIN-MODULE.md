# CSMFAB0116-54 — ULTRA-WIDEBAND FIN MODULE
## 1. SPECIFICATION
| Parameter | Value |
|-----------|-------|
| Chipset | NXP NCJ29D5 or Qorvo DW3300Q |
| Standard | IEEE 802.15.4z |
| Frequency | 6.5 GHz / 8.0 GHz (channel 5/9) |
| Bandwidth | 500 MHz |
| Antennas | 2 (AoA) or 3 (AoA+AoD) |
| Range (precise) | Up to 50m |
| Accuracy | ±10 cm (ranging), ±3° (angle) |
| Update rate | Up to 100 Hz |
| Power (ranging) | 50 mW |
| Power (idle) | 0.5 mW |

## 2. USE CASES
| Use Case | Description |
|----------|-------------|
| Digital Car Key | CCC Digital Key 3.0 |
| Precision finding | Find lost items with UWB tags |
| Spatial awareness | Detect nearby UWB devices |
| Secure access | Proximity-based door unlock |
| Gesture control | Hand tracking at short range |

## 3. SECURITY
| Feature | Description |
|---------|-------------|
| Scrambled Timestamp Sequence | Prevents replay attacks |
| PHY-layer encryption | AES-128 on payload |
| Secure ranging | Distance bounding protocol |
