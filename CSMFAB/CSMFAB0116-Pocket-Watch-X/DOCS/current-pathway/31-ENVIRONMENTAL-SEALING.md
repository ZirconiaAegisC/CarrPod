# CSMFAB0116-31 — ENVIRONMENTAL SEALING
## 1. IP68 DESIGN
| Location | Seal Type | Material | Durometer |
|----------|-----------|----------|-----------|
| Display bezel | Radial O-ring | FKM (Viton) | 70 Shore A |
| Case back | Face O-ring | FKM | 70 Shore A |
| Crown stem | Double O-ring + grease | Silicone | 50 Shore A |
| Fin slot covers | Compression gasket | Silicone | 40 Shore A |
| Mic port | Membrane | ePTFE (Gore-Tex) | — |
| Speaker port | Membrane | ePTFE | — |
| Pressure vent | Membrane | ePTFE | — |
| Divot pads | Nano-coating | Parylene-C, 5 µm | — |

## 2. IP68 TEST PROTOCOL
1. Visual inspection of all seals.
2. Pressure decay test (0.3 bar, < 0.01 bar drop over 30s).
3. Submersion: 2m depth, 30 minutes, powered on.
4. Post-submersion functional test.
5. Internal humidity indicator checked (must not trip).

## 3. NANO-COATING
| Parameter | Value |
|-----------|-------|
| Material | Parylene-C |
| Thickness | 5 µm |
| Coverage | All PCBs and connectors (masked: contacts, connectors) |
| Process | CVD (chemical vapor deposition) |
| IP effect | IPX3 without seals, IP68 with seals |
| Cost | $3/unit |

## 4. VERIFICATION
| Test | Method | Criterion |
|------|--------|-----------|
| IP68 | 2m, 30 min | No water ingress |
| Pressure decay | 0.3 bar | < 0.01 bar/30s |
| Thermal shock seal | -40 to +85°C, 100 cycles | IP68 retained |
| Vibration | Per MIL-STD-810H | IP68 retained |
