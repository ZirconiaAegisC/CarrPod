## V1.0 vs V2.0 Parameter Changes

| Parameter | V1.0 | V2.0 | Consequence if Not Updated |
|---|---|---|---|
| **Connector type** | Threaded PEEK (5 turns) | Bayonet-lock ZrO2 (quarter turn) | 8,000 thread engagements = 4-5hr assembly; bayonet = 7 min |
| **Segment cost** | $18/segment (commercial ZrB2) | $9.50/segment (in-house SHS ZrB2) | 47% cost reduction via Aegis-Forge I supply chain |
| **Deployment speed** | 1.2 ms per segment | 0.8 ms (MAX Phase hinge) | 33% faster full-rope deployment |
| **Operating temp range** | -50 to +200degC | -60 to +400degC | Extended for re-entry thermal scenario |
| **Quality control** | Manual inspection | LIBS auto-analysis (Aegis-Forge I) | Manual misses sub-surface ZrB2 phase defects |
