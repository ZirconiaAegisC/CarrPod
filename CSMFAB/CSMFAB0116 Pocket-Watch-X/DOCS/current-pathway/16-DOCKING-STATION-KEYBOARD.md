# CSMFAB0116-16 — POCKET WATCH X: DOCKING STATION — KEYBOARD
## 1. SPECIFICATION
The Keyboard Dock transforms the Pocket Watch X into a mini laptop experience.
| Parameter | Value |
|-----------|-------|
| Keyboard type | Mechanical, 68-key (ANSI 65% + 3 function keys) |
| Switch type | Kailh Choc V2 Low Profile (Brown tactile, 50g actuation) |
| Key travel | 3.2 mm |
| Actuation distance | 1.5 mm |
| Keycap material | PBT dye-sublimated |
| Backlight | Per-key RGB (16.8M colors) |
| Trackpad | Glass surface, 80 × 50 mm, haptic click |
| Battery | 40 Wh Li-Po (charges watch while docked) |
| Connectivity | BT 5.4 + USB-C wired mode |
| Dimensions | 280 × 130 × 12 mm (folded) |
| Weight | 650 g |
| Material | 6061 aluminum unibody + vegan leather base |

## 2. DOCK CONNECTOR
| Parameter | Value |
|-----------|-------|
| Watch slot | Center-top, 50mm well with divot connector |
| Retention | Magnetic ring (N52 NdFeB) + spring latch |
| I/O | 1× USB-C (data + PD passthrough), 1× HDMI 2.1 |
| Power to watch | 15W (Qi2 MPP) + 5W (wired via divot) |

## 3. OPERATION MODES
- **Desktop Mode:** Watch UI renders windowed desktop on keyboard dock integrated display (not included in base model — uses external monitor via HDMI).
- **Wireless Mode:** Keyboard connects via BT 5.4 to the watch. Watch remains in pocket/hand.
- **Standalone Mode:** Keyboard functions as standard BT keyboard for any device after initial pairing to watch.

## 4. VERIFICATION
| Test | Method | Criterion |
|------|--------|-----------|
| Keystroke lifespan | Robotic actuator, 50M clicks per key | 100% functional |
| BT latency | Key-to-display measurement | < 15 ms |
| Battery life | Full charge, continuous typing | > 80 hours |
| Trackpad accuracy | Windows Precision Touchpad test | Pass Level 2 |
