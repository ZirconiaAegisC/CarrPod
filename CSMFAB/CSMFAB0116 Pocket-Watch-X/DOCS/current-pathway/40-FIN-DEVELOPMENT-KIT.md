# CSMFAB0116-40 — FIN DEVELOPMENT KIT (FDK)
## 1. KIT CONTENTS
| Item | Description |
|------|-------------|
| FDK Base Board | Breakout board with fin slot connector, USB-C, JTAG |
| Reference CPU Fin | Open-source RISC-V CPU fin (SiFive FE310-based) |
| Reference Sensor Fin | Open-source 9-axis IMU fin |
| Fin Blank PCB | 6-layer HDI, 22×12mm, unpopulated |
| Fin Connector | 60-pin mezzanine (5 pcs) |
| Fin Housing Blank | 3D-printed plastic fin housing |
| Cables | USB-C, JTAG ribbon, UART |
| Documentation | FDK manual, API reference, schematics, layout files |

## 2. FDK SDK
| Component | Language | Description |
|-----------|----------|-------------|
| libfinbus | C | Low-level fin bus driver (PCIe enumeration, FDB read) |
| libfindrv | C | Fin driver template |
| fin-cli | Python | Command-line fin testing tool |
| fin-sim | QEMU-based | Fin emulation for development without hardware |
| fin-cert | Python | Pre-certification test suite |
