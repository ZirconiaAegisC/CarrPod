# CSMFAB0116-F19 — METAMATERIAL ANTENNA PATHWAY: Reconfigurable metamaterial-based antenna array without traditional phased array. Beamforming via tunable metamaterial elements (varactor-loaded split-ring resonators). Frequency agility: software-defined, single antenna covering 600 MHz — 40 GHz. Benefits: no bulky antenna array, full-band cellular + WiFi + GNSS from one compact element. Research: Kymeta (satellite), Echodyne (radar), academic metamaterial antenna groups. Timeline: 2028-2032 for mobile form factor. Impact: eliminates multiple antennas in Pocket Watch X.
