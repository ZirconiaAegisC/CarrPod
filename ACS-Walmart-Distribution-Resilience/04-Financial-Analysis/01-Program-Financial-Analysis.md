
# FINANCIAL ANALYSIS
## ACS-WALMART-DISTRIBUTION-RESILIENCE

**CLASSIFICATION: COMMERCIAL-IN-CONFIDENCE**

## 1. PROGRAM COST

| Phase | Cost |
|-------|------|
| Phase 0 — 10-DC assessment | $2.8M |
| Phase 1 — 10 priority DCs | $24.5M |
| Phase 2 — 150+ DCs substation | $215.0M |
| Phase 3 — 500 priority stores | $142.0M |
| Phase 4 — Fleet protonic network | $18.0M |
| **TOTAL CORE** | **$402.3M** |
| Phase 5 — Full store network (optional) | $2.1B |

## 2. LOSS PREVENTION (NATIONAL-SCALE EVENT)

| Loss Category | Cost |
|---------------|------|
| Food inventory loss (DCs) | $3.2-7.5B |
| Store perishable loss | $2.5-5.8B |
| Supply chain disruption (6-12 weeks) | $2.0-6.0B |
| Brand and reputational impact | $500M-1.5B |
| Community health impact (food/pharmacy) | $300-1.2B |
| **TOTAL RANGE** | **$8.5-22.0B** |

## 3. ROI

| Program investment | $402.3M |
| Loss prevented | $8.5-22.0B |
| ROI at first event | 21x - 55x |
| Walmart annual revenue | ~$680B |
| Program as % annual revenue (4 years) | ~0.015%/year |

## 4. PER-DC JUSTIFICATION

A single grocery DC holds $20-40M in perishable inventory. Protecting one DC costs ~$1.6M (substation NBC + refrigeration protection). The protection pays for itself at 4-8% of the first event loss prevented — for a single DC.
