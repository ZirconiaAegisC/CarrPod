# Verdagy — 20MW eDynamic Module Architecture

## Module Concept

The eDynamic 20MW electrolyzer module is the commercial building block of Verdagy's
product platform. Rather than selling individual electrolyzer stacks that must be
integrated by the customer or an engineering contractor, Verdagy delivers a fully
integrated, factory-assembled module that includes all subsystems required for hydrogen
production. This approach — analogous to the containerized battery energy storage
systems that transformed the grid storage industry — reduces site installation time,
minimizes integration risk, and enables standardized performance guarantees.

## Module Specifications

| Parameter | Specification |
|-----------|--------------|
| Nameplate capacity | 20 MW electrical input |
| Hydrogen output | ~4,000 Nm³/h (approximately 360 kg/h) |
| Module dimensions | Standard shipping container footprint (multiple containers) |
| Cell count per module | ~200-300 cells (depending on configuration) |
| Operating pressure | Up to 30 bar (pressurized option) |

| System efficiency | >80% (HHV basis, including auxiliary loads) |

## Subsystem Architecture

Each 20MW module integrates the following subsystems:

1. **Power Conditioning** — transformers and rectifiers that convert medium-voltage
   AC grid power to the low-voltage, high-current DC required by the electrolyzer
   stacks, including harmonic filtering for grid code compliance

2. **Electrolyzer Stacks** — multiple stacks arranged in parallel-series configuration
   to match the electrical input and hydrogen output requirements

3. **Electrolyte Management** — pumps, tanks, heat exchangers, and filtration systems
   that circulate, condition, and purify the potassium hydroxide electrolyte

4. **Gas Separation & Processing** — hydrogen-oxygen separators, demisters, and dryers
   that produce high-purity hydrogen suitable for industrial use or compression

5. **Thermal Management** — cooling systems that reject process heat to ambient air
   or available cooling water, maintaining optimal cell operating temperature

6. **Control & Safety** — programmable logic controllers, hydrogen sensors, ventilation
   interlocks, and emergency shutdown systems compliant with industrial gas standards

## Factory Assembly & Testing

A distinguishing feature of the eDynamic module is that it is fully assembled and
tested at Verdagy's Moss Landing factory before shipping to the customer site. Factory
acceptance testing includes:

- Full-power operation for a specified duration
- Hydrogen purity verification at multiple operating points
- Leak testing of all gas handling components
- Control system functional validation
- Safety system trip testing

This approach catches integration issues before they reach the field, reducing
commissioning time and improving first-year reliability.

## Modular Scaling

The 20MW module is designed for parallel deployment. A 100MW facility — such as
Verdagy's announced Texas project — comprises five 20MW modules operating in concert,
with shared balance-of-plant infrastructure for water treatment, hydrogen compression,
and storage. The modular architecture enables:

- **Phased capacity additions** — add modules as hydrogen demand grows
- **Standardized maintenance** — common spare parts and procedures across all modules
- **Technology upgradeability** — newer-generation modules can operate alongside
  existing units
- **Financing flexibility** — modular capital deployment matched to offtake agreements
