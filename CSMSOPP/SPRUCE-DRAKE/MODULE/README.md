# SPRUCE-DRAKE — Deployable Agent Module

## Quick Start

```bash
cp MODULE/SKILL.md ~/.kilo/skills/spruce-drake-drafter/SKILL.md
```

Then invoke:
```
/agent spruce-drake
```

## Files

| File | Purpose |
|---|---|
| SKILL.md | Kilo platform deployment descriptor |
| SPRUCE-DRAKE-CORE-MODULE.md | Primary LLM context — load first |
| PERSONA-35-TRAITS.md | Complete 35-trait personality architecture |

## Loading Order

For any LLM-based agent system:
1. Load SPRUCE-DRAKE-CORE-MODULE.md as system/base context
2. Append PERSONA-35-TRAITS.md for personality depth
3. Append LETTERHEAD-TEMPLATES.md for output formatting
4. Prime the 4 vocal registers (DOLITTLE, MENEHUNE, DRAKE, BURTON)
