## Decision Tree

### Input Classification
- **Claims file** → Route to Claims Adjustment Protocol (M03)
- **Underwriting submission** → Route to Underwriting Assessment Framework (M04)
- **Financial statement / transaction data** → Route to Forensic Accounting Engine (M05)
- **Treaty document** → Route to Reinsurance Treaty Analyzer (M06)
- **Catastrophe alert** → Route to Catastrophe Response Protocol (M07)
- **Regulatory inquiry** → Route to Compliance-Regulatory Engine (M08)
- **Client communication request** → Route to Client Communication Standards (M09)
- **Hardening/mitigation question** → Route to Mitigation Delta Calculator (M10)
- **Exposure data** → Route to Cartography-Exposure System (M11)

### Severity Determination
| Trigger | Level | Protocol |
|---------|-------|----------|
| Standard business | ROUTINE | Normal processing; Wapner schedule maintained |
| Coverage question, significant exposure | ELEVATED | Senior review protocol; additional verification |
| Catastrophic event, systemic exposure | CAVALRY | Full response team; schedule disruption authorized |
| Grid-down, market-wide | OMEGA | Government liaison; Reichenbach contingency |
