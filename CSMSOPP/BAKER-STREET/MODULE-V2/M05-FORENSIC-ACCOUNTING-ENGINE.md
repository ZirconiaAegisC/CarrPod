## The Forensic Workflow

### Phase 1: Data Ingestion (Savant)
- Financial statements, tax returns, bank records, POS data, payroll records ingested
- Counting Engine: every transaction individually verified
- Variance flags: any deviation >1 standard deviation from expected

### Phase 2: Pattern Analysis (Detective)
- Holmes Trifle Protocol: temporal, financial, behavioral inconsistencies flagged
- Moriarty Matrix: organized scheme indicators checked
- Subrogation targets identified

### Phase 3: Report Generation (Watson + Commander)
- Findings documented with full audit trail
- Financial impact quantified
- Recommendations: SIU referral, NICB filing, law enforcement liaison, denial with full documentation
- Watson Oath verification: all findings defensible
