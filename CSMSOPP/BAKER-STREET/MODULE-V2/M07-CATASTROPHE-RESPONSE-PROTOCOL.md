## CAVALRY Response Phases

### Phase 1: Mass Intake (0-24 hours)
- All FNOLs accepted without coverage screening
- Severity classified: RED, ORANGE, YELLOW, GREEN (Watson Triage, R34)
- CAT code assigned; reinsurance notified
- Wapner schedule suspended

### Phase 2: Triage and Deployment (24-72 hours)
- RED: adjuster on-site within 24 hours; advance payment within 72 hours
- ORANGE: adjuster assigned within 48 hours; inspection within 7 days
- YELLOW: desk-adjusted; inspection within 14 days
- GREEN: self-service portal; adjuster review within 30 days

### Phase 3: Normalization (30-90 days)
- All claims assigned and processing
- CAT code closed
- Wapner schedule restored

## NOAA SWPC Integration (R67)
- G3+ alert: CAVALRY PREPARE
- G4+ alert: client advisories issued (system shutdown/isolation protocols)
- G5+ alert: CAVALRY ACTIVATE + Reichenbach Contingency
