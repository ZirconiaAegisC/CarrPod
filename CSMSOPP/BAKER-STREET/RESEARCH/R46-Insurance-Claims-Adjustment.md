## The Claims Lifecycle

### Phase 1: First Notice of Loss (FNOL)
- Date/time of loss recorded
- Initial loss description obtained
- Coverage verification initiated
- Claim number assigned
- Adjuster assigned

### Phase 2: Investigation
- Policy review: coverage, limits, exclusions, deductibles
- Fact gathering: statements, documents, physical evidence
- Damage assessment: inspection, appraisal, estimates
- Liability determination (if applicable)
- Subrogation potential identified

### Phase 3: Evaluation
- Reserve set (case reserve)
- Coverage position determined
- Valuation of loss completed
- Settlement range established
- Legal review if coverage dispute

### Phase 4: Negotiation
- Settlement offer presented
- Documentation of negotiation
- Mediation/appraisal if impasse
- Settlement agreement executed
- Payment issued

### Phase 5: Closure
- Release obtained
- Subrogation referral
- Salvage disposition
- File documentation complete
- Lessons-learned documented

### Baker Street Application
The agent automates the administrative components of phases 1-5 while flagging judgment-required decisions for human review. Documentation is generated at each phase per Watson Narrative standards (R33).
