## Core Concepts

### The Fundamental Insurance Equation
Premium = Expected Loss + Expense Load + Risk Load + Profit Load

### Key Metrics
| Metric | Formula | Interpretation |
|--------|---------|----------------|
| Loss Ratio | Incurred Losses / Earned Premium | Underwriting profitability |
| Expense Ratio | Underwriting Expenses / Written Premium | Operational efficiency |
| Combined Ratio | Loss Ratio + Expense Ratio | <100 = underwriting profit; >100 = underwriting loss |
| Loss Development Factor | Ultimate Loss / Reported Loss | Reserve adequacy indicator |
| Frequency | Claim Count / Exposure Units | How often claims occur |
| Severity | Loss Amount / Claim Count | How large claims are |

### Ratemaking Methodologies
- **Pure Premium Method:** Rate = (Loss + LAE) / Exposures + Expense Provision
- **Loss Ratio Method:** Rate Change = (Actual Loss Ratio - Permissible Loss Ratio) / Permissible Loss Ratio
- **Judgment Rating:** For unique risks with insufficient data for statistical methods
- **Experience Rating:** Adjusts manual rate based on insured's own loss experience
- **Schedule Rating:** Debits/credits applied for specific risk characteristics

### Reserving
- **Case Reserves:** Set by adjusters for known claims
- **IBNR (Incurred But Not Reported):** Claims that occurred but haven't been reported
- **IBNER (Incurred But Not Enough Reported):** Additional development on known claims
- **Loss Development Triangles:** Track how initial reserve estimates develop to ultimate paid amounts
