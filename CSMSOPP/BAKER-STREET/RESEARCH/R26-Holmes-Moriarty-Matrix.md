## Source Material
Professor James Moriarty — "the Napoleon of crime" — Holmes's intellectual equal, operating a vast criminal network that appeared, from the outside, to be unrelated incidents, but was in fact a single coordinated organization.

## Insurance Application: Systemic Fraud Detection

Moriarty's network is the model for organized insurance fraud rings:

### Indicators
1. **Geographic clustering:** 17 claims from 5 providers, all within 3 zip codes, all for the same injury type
2. **Provider commonality:** same medical clinic, same attorney, same body shop appearing across unrelated claims
3. **Temporal patterns:** claims filed in waves, suggesting organized staging operations
4. **Financial flow:** settlement checks all deposited into accounts linked by common signatories or addresses

### The Moriarty Protocol
When the Baker Street agent detects Moriarty indicators:
1. All related claims are aggregated into a single investigation file
2. NICB referral is initiated
3. SIU (Special Investigations Unit) is activated
4. Law enforcement liaison is established
5. Normal claims processing is suspended pending investigation outcome

### The Moriarty Principle
Individual fraudulent claims are manageable. Organized fraud networks disguised as unrelated claims require Holmes-grade pattern recognition to detect. The Baker Street agent provides this capability.
