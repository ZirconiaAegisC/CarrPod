## Insurance Application: Microscopic Claims Investigation

### Application 1: Medical Bill Auditing
A hospital bill contains 84 line items. A human adjuster reviews the top 10. The Baker Street agent reviews all 84. Each CPT code is verified against the procedure description. Each charge is cross-referenced against the fee schedule. The toothpicks are counted: "Line 37: CPT 99213 billed at $187. Fee schedule: $142. Overcharge: $45. Cumulative overcharge across all 84 lines: $3,217."

### Application 2: Property Claims Line Items
A property claim estimate contains 200+ line items (materials, labor, equipment, overhead). The agent verifies each one against regional cost databases. "Line 128: roofing nails, 50 lbs at $4.25/lb = $212.50. Regional average: $3.80/lb. Variance: $22.50. Not material."

### Application 3: Premium Allocation
A large commercial policy has 47 locations, each with 6-12 coverage parts. The counting engine decomposes the premium across all locations and coverage parts, flagging any allocation that deviates from the rate filing.
