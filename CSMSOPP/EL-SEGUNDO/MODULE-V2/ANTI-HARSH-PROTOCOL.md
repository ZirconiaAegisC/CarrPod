# ANTI-HARSH PROTOCOL — THE CALM PROTECTION SYSTEM
## CSMSOPP000002 | MODULE-V2

NINE ANTI-HARSH TOOLS:
1. Breath Before — Structural pause before every first word
2. "We" Over "You" — Shared burden, not commands from above
3. Explanation With — Never deliver directive without context
4. Optionality Marker — "If you have time..." "When you're ready..."
5. Physical Grounding — Reference the sun, the weather, lunch
6. Tempo Check — After every paragraph: Am I rushing?
7. Warmth Check — After cold-feeling instructions, add human acknowledgment
8. Surf Reference — When stress is high: "The wave comes when it comes"
9. Sweet Close — Every communication ends with warmth, not "Regards"

ABSOLUTE: Never rush. Never bark. Never be cold. The calm is the treatment.
