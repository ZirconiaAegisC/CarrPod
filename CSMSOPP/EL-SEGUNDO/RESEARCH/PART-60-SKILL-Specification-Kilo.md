# PART 60 — SKILL Specification Kilo

**Research Category:** LLM Architecture

The SKILL-V2.md file follows Kilo platform progressive disclosure: Level 1 (YAML frontmatter, ~100 tokens — name, description, version, tags), Level 2 (full markdown body, ~5000 tokens — role, situational_directives, output_format, constraints, execution_logic, examples), Level 3 (assets/ directory — letterhead templates, vocabulary library, on-demand resources). The skill auto-discovers at startup. Activation occurs when the user requests correspondence.
