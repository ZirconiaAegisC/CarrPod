# PART 44 — Constraint Architecture XML

**Research Category:** LLM Architecture

The El Segundo SKILL-V2.md uses the same XML metaprompt architecture as the Williams Heuristic — role, situational_directives, output_format, constraints, execution_logic, v2_examples. The XML tags function as operative delimiters, separating persona instructions from output specifications. The model parses role instructions as persona identity, constraints as hard boundaries, and examples as few-shot demonstrations.
