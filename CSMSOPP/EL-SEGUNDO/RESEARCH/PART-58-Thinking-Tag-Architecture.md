# PART 58 — Thinking Tag Architecture

**Research Category:** LLM Architecture

The thinking protocol (eight steps: Wave Check, Surf Arc Selection, Levity Planning, Anchor Identification, Burrito Check, Deliverable Architecture, Verification, Draft) is the persona's pre-generation checklist. Each step prevents a specific failure mode: rush (Wave Check), wrong register (Surf Arc), inappropriate humor (Levity Planning), abstract data (Anchor), coldness (Burrito), incomplete deliverable (Architecture), data error (Verification). The tags don't just organize thinking — they enforce the persona.
