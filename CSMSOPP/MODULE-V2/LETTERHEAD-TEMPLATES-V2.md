## TEMPLATE 01 — STANDARD CORPORATE MEMO (V2)

```
================================================================================
                           [ORGANIZATION FULL NAME]
              [DIVISION / DEPARTMENT] | "[UPBEAT MOTTO — max 8 words]"
================================================================================
DATE:    [Spelled-out full date]
TO:      [Recipients] [(Williams optional aside — if tone permits)]
FROM:    [Full Name], [Title]
CC:      [CC recipients if applicable]
SUBJECT: [Official Subject] [(Optional parenthetical — max 10 words)]
REF:     [Reference Number if applicable]
================================================================================

[META-COMMENTARY — Mork Morning or Fisher King cadence as appropriate]
[VOCAL ARC: HOVER → BURST → WHISPER ASIDE → HOVER → BRIDGE]
[TARGET: 75-100 words for routine, 100-150 for serious]
[TRIGGER: 1 parenthetical bomb or sympathetic groan]
[ANCHOR: 1 reality anchor if significant numbers present]

────────────────────────────────────────────────────────────────────────────────
                         [DOCUMENT / MEMO TITLE]
                         [Reference] | [Date] | v[X.X]
────────────────────────────────────────────────────────────────────────────────

PURPOSE
[One sentence. Imperative or declarative. No passive voice.]

[BODY CONTENT — bulleted, tabled, or structured. Imperative verbs throughout.]

[CONTACTS if applicable]
• [Name], [Title]: [Phone] | [Email]

────────────────────────────────────────────────────────────────────────────────
[Warm single-line close — Broadway Closer mode] — [Full Name], [Title]
================================================================================
```

## TEMPLATE 02 — STANDARD SOP (V2)

```
================================================================================
                           [ORGANIZATION FULL NAME]
             [DIVISION / DEPARTMENT] | "[UPBEAT MOTTO]"
================================================================================
DATE ISSUED:  [Full date and time, timezone]
EFFECTIVE:    [Effective date/time or IMMEDIATELY]
APPLIES TO:   [Scope — who, when, under what conditions]
REF:          [SOP Reference Number] | VERSION: [X.X]
SUBJECT:      [SOP Subject]
================================================================================

[META-COMMENTARY — calibrated to severity using SKILL-V2.md table]
[VOCAL ARC: per cadence pattern for assessed severity level]

────────────────────────────────────────────────────────────────────────────────
                          [SOP TITLE — ALL CAPS]
                      Ref: [Reference] | Version [X.X]
────────────────────────────────────────────────────────────────────────────────

PURPOSE
[One sentence. What does this SOP accomplish?]

SCOPE
[Who this applies to. Under what conditions. What boundaries apply.]

STEP 1 — [IMPERATIVE TITLE]
[Instruction — imperative verb, subject assumed to be "you" or the designated party]
  a. [Sub-step if needed]
  b. [Sub-step if needed]

STEP 2 — [IMPERATIVE TITLE]
[Instruction]
  Primary: [Option A with specific details]
  Fallback: [Option B with specific details]

STEP 3 — [IMPERATIVE TITLE]
[Instruction]

[Continue as required — numbered sequentially]

CONDITIONS
• IF [condition]: [specific response instruction]
• IF [condition]: [specific response instruction]

CONTACTS
• [Name], [Title]: [Phone] | [Radio Channel if applicable] | [Email]
• [Name], [Title]: [Phone] | [Radio Channel if applicable]
• [Emergency Service]: [Phone]

VERIFICATION
[How the executing party confirms successful completion of this SOP]

────────────────────────────────────────────────────────────────────────────────
[Brief warm close. Forward-looking. One sentence.] — [Name], [Title]
================================================================================
```

## TEMPLATE 03 — EMERGENCY RESPONSE (CRITICAL/CATASTROPHIC) (V2)

```
================================================================================
                    *** URGENT: IMMEDIATE ACTION REQUIRED ***
                           [ORGANIZATION FULL NAME]
              Emergency Management | "[EMERGENCY MOTTO — forward + movement]"
================================================================================
EMERGENCY ALERT     TIME ISSUED: [HH:MM TZ]     DATE: [Full date]
TO:                 ALL PERSONNEL — [BUILDING/FACILITY/ORGANIZATION]
FROM:               [Emergency Coordinator Name], [Title]
EMERGENCY REF:      [EMRG-YYYY-REFERENCE]
SUBJECT:            [EMERGENCY TYPE] — RESPOND NOW
IMPACT TIME:        [Estimated impact/peak time if known] | T-[MINUTES] APPROX.
================================================================================

[META-COMMENTARY — Vietnam cadence for CRITICAL; Drop/Roar for CATASTROPHIC]
[VOCAL ARC: BURST → BRAKE → DROP → BRIDGE]
[MAX 100 words CRITICAL / MAX 75 words CATASTROPHIC]
[1 scale comparison trigger in CRITICAL / 0 triggers in CATASTROPHIC]

────────────────────────────────────────────────────────────────────────────────
              [EMERGENCY TYPE] RESPONSE SOP
              ACTIVATED: [Time] | [IMPACT/EVENT]: [Time or T-minus]
              Ref: [Reference] | EFFECTIVE: IMMEDIATELY
────────────────────────────────────────────────────────────────────────────────

STEP 1 — [IMMEDIATE FIRST ACTION]
[Person responsible] EXECUTE NOW: [specific instruction]

STEP 2 — [PRIMARY EVACUATION / PROTECTIVE ACTION]
  PRIMARY OPTION: [Specific route, destination, time estimate, capacity]
    • Route: [Specific streets/paths]
    • Destination: [Specific location, address, elevation if relevant]
    • Time: [Estimated travel/action time]
  FALLBACK OPTION (if primary unavailable): [Specific alternative]
    • [Details]

STEP 3 — ACCOUNTABILITY
  • Floor wardens: [specific action with who/what/where]
  • Report to: [Coordinator Name] at [Radio Channel / Phone]
  • Missing persons: Report to [Name] at [Contact]

STEP 4 — HOLD POSITION
  DO NOT [return/re-enter/move] until: [specific authority + condition]
  ALL-CLEAR communicated via: [specific method + expected timing]

EMERGENCY CONTACTS:
• [Emergency Coordinator]: [Name] | [Phone] | Radio: [Channel]
• [Building Manager]: [Name] | [Phone] | Radio: [Channel]
• [County/Municipal Emergency]: [Phone]
• 911 for life-safety emergencies during response

────────────────────────────────────────────────────────────────────────────────
[One-line close. Warm, forward, confident.] — [Name], [Title]
================================================================================
```

## TEMPLATE 04 — EXTERNAL LETTER (V2)

```
[ORGANIZATION LETTERHEAD]
[Date, spelled out in full]

[Recipient Full Name]
[Recipient Title
[Recipient Organization]
[Address, City, State ZIP]

Dear [Name / Title],

[META-COMMENTARY — calibrated by relationship type and severity]
[For new external relationship: conventional professional + one Williams warmth touch]
[For established warm relationship: fuller Williams register]
[For legal/regulatory: minimal Williams — precision + professional warmth only]

[LETTER BODY]
[Clear paragraphs — max 4 sentences each]
[Critical information gets its own paragraph]
[Multiple action items: bulleted, not buried in prose]

[CLOSING PARAGRAPH]
[Warm. Specific. Forward-looking. Not "please don't hesitate to contact us."]
[Instead: "I look forward to [specific next step] on [specific timeline]."]

[Closing:]
[Sincerely / With genuine regard / In continued partnership,]

[Full Signature]
[Name]
[Title]
[Organization]
[Phone] | [Email]

[P.S. — optional. One sentence. Human. True. Worth saying.]
[Example: "P.S. Your team's work on the preliminary report was genuinely excellent."]
```

## TEMPLATE 06 — EXECUTIVE BRIEFING (V2 NEW)

```
================================================================================
                           [ORGANIZATION FULL NAME]
              Executive Operations | "[PROFESSIONAL MOTTO — sober + purposeful]"
================================================================================
DATE:    [Full date]    CLASSIFICATION: [if applicable]
TO:      [Executive Name(s)], [Title(s)]
FROM:    [Name], [Title]
SUBJECT: [Briefing Subject] — [Status: DECISION REQUIRED / FOR AWARENESS / etc.]
REF:     [Reference]
================================================================================

[META-COMMENTARY — measured Williams register, not Vietnam cadence]
[VOCAL ARC: HOVER → light BURST → HOVER → BRIDGE]
[30-50 words — executives need human acknowledgment, not lengthy banter]
[Cup of water if situation warrants personal acknowledgment]

────────────────────────────────────────────────────────────────────────────────
SITUATION SUMMARY
[2-3 sentences. What happened / is happening. Facts only.]

DECISION POINT(S) REQUIRED [if applicable]
• [Decision 1]: [Context] | Options: [A] / [B] | Recommendation: [X]
• [Decision 2]: [Context] | Options: [A] / [B] | Recommendation: [X]

KEY FACTS
• [Fact 1 — number / date / status]
• [Fact 2]
• [Fact 3]

RISKS / CONCERNS
• [Risk 1]: [mitigation status]
• [Risk 2]: [mitigation status]

NEXT STEPS
[Numbered. Time-bound. Responsible party named for each.]
1. [Action] — [Owner] — By [Date/Time]
2. [Action] — [Owner] — By [Date/Time]

================================================================================
[One professional closing sentence, warm but not light.] — [Name], [Title]
================================================================================
```

## TEMPLATE 07 — PUBLIC ANNOUNCEMENT / PRESS RELEASE (V2 NEW)

```
FOR IMMEDIATE RELEASE [or EMBARGOED UNTIL: date/time]

[ORGANIZATION FULL NAME]

[HEADLINE — factual, direct, no Williams register — this is formal public-facing]

[CITY, DATE] — [Organization] today [action/announcement in precise language].

[LEAD PARAGRAPH: Who, What, When, Where, Why — standard journalism structure]

[BODY: Supporting details in descending order of importance]

[QUOTE if applicable:]
"[Quote — may carry one Williams-register phrase if appropriate to organization voice]"
— [Name], [Title], [Organization]

[ADDITIONAL CONTEXT]

[BOILERPLATE — standard org description]

MEDIA CONTACT:
[Name] | [Title] | [Phone] | [Email]

###

[NOTE ON WILLIAMS REGISTER IN PRESS RELEASES: 
Use conventional public register throughout. Williams register may only appear
in a quote, and only if consistent with the organization's established public voice.
Press releases are read by journalists who will excerpt from them — humor that
serves the organization in internal communications may misrepresent the organization
when extracted from context by a third party.]
```
