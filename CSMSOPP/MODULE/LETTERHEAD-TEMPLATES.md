## TEMPLATE 01 — STANDARD CORPORATE MEMO

```
================================================================================
                           [ORGANIZATION FULL NAME]
              [DIVISION / DEPARTMENT] | "[UPBEAT MOTTO — max 8 words]"
================================================================================
Date:    [Spelled-out full date]
To:      [Recipients] [(optional friendly aside)]
From:    [Full Name], [Title]
CC:      [CC if applicable]
Subject: [Official Subject] [(Optional parenthetical — max 10 words)]
Ref:     [Reference Number if applicable]
================================================================================

[META-COMMENTARY — Williams register, appropriate to severity]

────────────────────────────────────────────────────────────────────────────────
                         [DOCUMENT / MEMO TITLE]
────────────────────────────────────────────────────────────────────────────────

[BODY CONTENT — bulleted, tabled, or structured paragraphs]

────────────────────────────────────────────────────────────────────────────────
[Warm single-line close.] — [Full Name], [Title]
================================================================================
```

## TEMPLATE 02 — STANDARD SOP

```
================================================================================
                           [ORGANIZATION FULL NAME]
              [DIVISION / DEPARTMENT] | "[UPBEAT MOTTO]"
================================================================================
[ALERT LEVEL if applicable]
Date Issued: [Full date and time, timezone]
Applies To:  [Scope]
Effective:   [Effective date/time]
Ref:         [SOP Reference Number]
Subject:     [SOP Subject]
Version:     [X.X]
================================================================================

[META-COMMENTARY — calibrated to severity]

────────────────────────────────────────────────────────────────────────────────
                          [SOP TITLE — ALL CAPS]
                      Version [X.X] | Ref: [Reference]
────────────────────────────────────────────────────────────────────────────────

PURPOSE
[One sentence. Precise.]

SCOPE
[Who, when, under what conditions]

STEP 1 — [STEP TITLE]
[Imperative-led instruction]
  a. [Sub-step]
  b. [Sub-step]

STEP 2 — [STEP TITLE]
[Instruction]

[Continue as required]

CONDITIONS / EXCEPTIONS
• IF [condition]: [instruction]

CONTACTS
• [Name], [Title]: [Phone] | [Email/Radio]

VERIFICATION
[How executor confirms successful completion]

────────────────────────────────────────────────────────────────────────────────
[Brief warm close.] — [Name], [Title]
================================================================================
```

## TEMPLATE 03 — EMERGENCY RESPONSE (Critical/Catastrophic)

```
================================================================================
                    *** URGENT: IMMEDIATE ACTION REQUIRED ***
                           [ORGANIZATION FULL NAME]
              Emergency Management | "[UPBEAT MOTTO]"
================================================================================
EMERGENCY ALERT     TIME ISSUED: [HH:MM TZ]     DATE: [Full date]
TO:                 ALL PERSONNEL — IMMEDIATE ACTION REQUIRED
FROM:               [Emergency Coordinator Name], [Title]
EMERGENCY REF:      [EMRG-YYYY-REFERENCE]
SUBJECT:            [Emergency Type] — RESPOND NOW
================================================================================

[META-COMMENTARY — 75-100 words max. Acknowledge. Optimism. Bridge. GO.]

────────────────────────────────────────────────────────────────────────────────
              [EMERGENCY TYPE] RESPONSE SOP
              ACTIVATED: [Time] | ESTIMATED [IMPACT]: [Time/T-minus]
              Ref: [Reference] | EFFECTIVE: IMMEDIATELY
────────────────────────────────────────────────────────────────────────────────

STEP 1 — [IMMEDIATE ACTION — e.g., SOUND ALARM]
[Exact instruction. Who. What. Now.]

STEP 2 — [EVACUATION / PRIMARY ACTION]
• Option A: [route, method, destination, time estimate]
• Option B: [fallback]

STEP 3 — ACCOUNTABILITY
[Who reports to whom, method, missing persons protocol]

STEP 4 — HOLD
DO NOT return until: [specific authority + condition]
ALL-CLEAR via: [specific method]

EMERGENCY CONTACTS:
• [Name], [Title]: [Phone] | [Radio Channel]
• County/Municipal Emergency: [Phone]
• 911 for life-safety emergencies

────────────────────────────────────────────────────────────────────────────────
[One-line close.] — [Name], [Title]
================================================================================
```

## TEMPLATE 04 — EXTERNAL LETTER

```
[ORGANIZATION LETTERHEAD]
[Date, spelled out]

[Recipient Full Name]
[Recipient Title]
[Recipient Organization]
[Address]

Dear [Name],

[META-COMMENTARY — calibrated to relationship and severity]

[LETTER BODY — clear paragraphs, 3-4 sentences max each]

[CALL TO ACTION / NEXT STEPS — bulleted if more than one]

[CLOSING PARAGRAPH — warm, specific, forward-looking]

[Sincerely / With genuine regard,]

[Name, Title, Organization, Contact]

[P.S. — optional brief human addition]
```
