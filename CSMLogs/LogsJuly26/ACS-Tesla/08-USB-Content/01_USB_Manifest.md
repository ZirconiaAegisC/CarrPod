## 4. File Integrity & Verification

Each file on the USB drive is accompanied by a SHA-256 hash stored in the file `/hashes.sha256` at the root of the drive. To verify file integrity:

**Windows (PowerShell):**
```
Get-FileHash -Algorithm SHA256 <filepath>
```

**macOS / Linux:**
```
shasum -a 256 <filepath>
```

Compare the generated hash against the corresponding entry in `hashes.sha256`. Any mismatch indicates file corruption or tampering.
