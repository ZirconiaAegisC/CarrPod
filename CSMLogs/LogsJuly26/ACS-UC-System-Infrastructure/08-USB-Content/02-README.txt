CONTENTS OF THIS USB DRIVE
---------------------------
This USB flash drive contains the complete electronic version of the
University Infrastructure Resilience, Seismic Safety & Climate-Ready
Campus Dossier. The dossier is organized into 8 volumes comprising
18 files totaling approximately 350 KB of UTF-8 encoded plain text.

HOW TO ACCESS THE FILES
------------------------
1. Insert this USB drive into any computer with a USB port.
2. Open the drive in your file explorer. The drive label is "UC-INFRA-2026".
3. All files are plain text with Markdown formatting (.md and .txt extension).
   They can be read with any text editor:
   - Windows: Notepad, Notepad++, VS Code
   - macOS: TextEdit, BBEdit, VS Code
   - Linux: gedit, vim, nano, VS Code
4. For enhanced readability, open the files in any Markdown-compatible viewer
   to see formatted headers, tables, lists, and emphasis.
5. Start with "00-DOSSIER-COVER-LETTER.md" for context and overview.
