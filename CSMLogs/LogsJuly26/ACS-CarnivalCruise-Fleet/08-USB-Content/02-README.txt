HOW TO USE
====================================

1. Start with DOSSIER/ACS-CCL-FP-Full-Dossier.pdf for the complete
   program document in final-form presentation format.

2. FINANCIAL-MODEL/ contains the interactive spreadsheet. Open the
   .xlsx file in Microsoft Excel (Office 365 or Excel 2019+). The
   model supports scenario analysis via the Assumptions worksheet.

3. DATA/ files are provided in Excel (.xlsx) and CSV formats for
   import into BI or analytics tools.

4. REFERENCES/ contains full-text research publications. PDF files
   require a PDF reader (Adobe Acrobat Reader or equivalent).

5. PRESENTATION/ contains a PowerPoint slide deck for executive
   briefings. Presenter notes are included on most slides.
