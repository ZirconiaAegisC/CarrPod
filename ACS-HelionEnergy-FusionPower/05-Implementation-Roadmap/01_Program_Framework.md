**Date:** 15 July 2026
**Classification:** Corporate Dossier — Section 05

## Framework Architecture

The Helion Energy program is structured across five concurrent workstreams, each with defined milestones, dependencies, and risk profiles. The program framework is designed to transition the company from prototype validation (Polaris) through first commercial operation (Orion) to serial plant deployment (Nucor and beyond).

## Critical Path Analysis

### Current Critical Path (July 2026)

```
Polaris D-He3 Campaign → Orion Generator Design Freeze → Generator Building Construction → Fusion Core Installation → Integrated Commissioning → Grid Synchronization → Commercial Operation
```

**Bottleneck Identification:**
1. **Chelan County PUD transmission interconnection agreement** — regulatory process, not under Helion's unilateral control.
2. **Generator building construction** — weather-dependent, supply-chain-dependent, and requires design freeze from Polaris data.
3. **Integrated commissioning** — first-of-kind; schedule contingency is inherently uncertain.

### Schedule Risk Buffer

The period between generator building completion (2027) and the Microsoft PPA deadline (2028) provides approximately 12–18 months of integrated commissioning time. For a first-of-kind fusion power plant, this is a compressed schedule with limited contingency.
