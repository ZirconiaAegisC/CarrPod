- This USB drive is AES-256 hardware-encrypted.
- Password is communicated separately via secure channel.
- After 5 incorrect password attempts, the USB drive will
  permanently lock and all data will be unrecoverable.
- DO NOT copy USB contents to network-accessible storage
  without prior authorization from ACS Document Control.

============================================================
 TROUBLESHOOTING
============================================================

Q: Drive does not appear after entering password.
A: Try a different USB port. Some ports may not supply
   sufficient power. If problem persists, contact ACS
   Document Control via secure channel.

Q: Cannot open Markdown files.
A: Files are plain text. Rename extensions to .txt and
   open with any text editor.

Q: File appears truncated or unreadable.
A: Verify file against SHA-256 checksums in the USB Master
   Manifest (08-USB-Content/01-USB-Master-Manifest.md).
   Contact ACS Document Control for reference hashes.
