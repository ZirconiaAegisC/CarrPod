## Subject: Beyond Life Safety: CSM's New Civil Infrastructure Resilience Standards Target Functional Continuity

**TO:** editor@cenews.com

CE News editors,

ASCE 7-22 and the IBC are designed for life safety — preventing collapse — not for functional continuity. A substation building that survives an earthquake without collapsing but loses its relay panels to interior acceleration has satisfied the building code but failed its mission. CIRS v2.0 closes this gap for critical infrastructure.

Key design criteria changes civil engineers need to know:
- 500-year flood elevations: +1.2m above current FEMA FIRM levels for sea-level rise through 2075
- ASCE 7 wind speeds: +8% in hurricane-prone regions based on NOAA observed intensity trends
- Seismic design spectra: 2,475-year return period ground motions (vs. 975-year standard) using USGS 2025 NSHM
- Functional-continuity performance objectives beyond conventional life-safety design
- Interdependent-system failure analysis requiring compound event modeling (seismic + flood + wind)
- 6.2:1 cost-benefit ratio — CSM Accountant Heuristic analysis
- Published open-access through ASCE standards portal with ICC code adoption pathway
- Applied to 4,700 assets across CSM's active portfolio
