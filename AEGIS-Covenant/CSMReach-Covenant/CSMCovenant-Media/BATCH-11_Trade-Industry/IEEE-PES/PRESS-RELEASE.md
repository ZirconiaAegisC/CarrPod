### CSM Treaty Publishes Open-Source Reference Architecture for Synthetic Inertia Deployment: IEEE PES Technical Committee Review Underway

**PISCATAWAY, NJ** — The Critical Systems Mandate today published the CSM Synthetic Inertia Reference Architecture (SIRA v2.1), a comprehensive open-source technical specification for deploying grid-forming inverter and synchronous condenser systems at transmission scale, developed from operational data across CSMFAB installations totaling 17.4 GW of deployed synthetic inertia capacity.

The SIRA v2.1 specification has been submitted to the IEEE PES Power System Stability Controls Subcommittee for technical review and provides the first publicly available reference architecture validated against real-world frequency event data from deployments in the ERCOT, PJM, ENTSO-E, and Australian NEM synchronous zones.

"This reference architecture closes the gap between academic research on grid-forming inverters and the practical engineering specifications that transmission planners require for interconnection studies," explained Dr. Thomas Park, CSM Chief Grid Architecture Engineer and IEEE Fellow. "Every parameter — from virtual impedance settings to fault ride-through curves to black-start sequencing logic — is published with the operational data that validates it, enabling independent peer review and adaptation to specific grid conditions."

The SIRA v2.1 specification covers: grid-forming inverter control architecture with detailed inner-loop current control and outer-loop voltage/frequency regulation parameters validated against IEEE 1547-2018 and IEEE 2800-2022 requirements; synchronous condenser deployment specifications including inertia constant (H) selection methodology, excitation system tuning for weak-grid applications, and coordination logic with existing generator protection systems; black-start and islanded operation procedures validated through CSMFAB field tests; and cyber-physical security architecture with post-quantum cryptographic authentication for all control communications.

CSM's open-architecture mandate requires that SIRA v2.1 be published under Creative Commons Attribution 4.0 International License, with all simulation models released as open-source PSCAD and PowerWorld cases. The IEEE PES technical committee review is expected to be completed by Q1 2027, with formal incorporation into IEEE recommended practice documentation to follow.
