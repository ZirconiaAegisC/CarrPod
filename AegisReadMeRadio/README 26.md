CSMApps — Application Development for Electromagnetic Resilience

The Software That Keeps Working When the Grid Goes Dark. Featuring Mork, Chester, and Zirconia.

THE QUICK PITCH

When the grid collapses, the apps on your phone become paperweights — unless they are designed to run offline, on hardened devices, through a self-healing mesh network. CSMApps develops Marmalade, the consumer-facing CarrPod control and emergency preparedness platform, and Landolil, the industrial-grade infrastructure monitoring and hardening verification tool. Both share an offline-first architecture that assumes disconnection. When SiblingFrequency Episode 2 features a civilian checking her Marmalade tablet during the grid collapse — "The battery says seventy-one hours. The water is at ninety percent. The mesh shows my sister's CarrPod three blocks away." — that interface is spec'd in this directory. Step into the dev studio.

THE CONVERSATION

A software development studio. Monitors display code editors and LoRa mesh topology maps. Mork is typing furiously. Chester is slouched in a beanbag chair, swiping through the Marmalade UI with unhurried rhythm. Zirconia stands at the architecture whiteboard.

ZIRCONIA: Welcome to CSMApps. I am Zirconia, director of software engineering, and these are Mork and Chester. The software industry has traditions stretching from Grace Hopper's compiler to the open-source movement. CSMApps builds on those traditions — offline-first architecture, encrypted local storage, peer-to-peer mesh networking. Mork, Marmalade.

MORK: (looks up from the keyboard) Marmalade is the CarrPod interface: environmental controls, preparation checklist, inventory tracker, family mesh communication, cached reference library, and a "What If" simulator for practicing response decisions. Everything works offline. The app syncs when it can but never depends on the sync. Offline-first is the software equivalent of the Dielectric Citadel.

CHESTER: Landolil is the professional side: GIC sensor monitoring, dielectric installation logging, premium credit reporting. It runs on a hardened tablet inside a Faraday-cage enclosure because a monitoring app that cannot survive the event it monitors is a liability. Right on.
